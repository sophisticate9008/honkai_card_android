# honkai_card_app
崩坏三星与你消失之日卡牌小游戏简单复刻，三选一将选择的卡牌拖到下面空位排序，支持匹配模式，但是先手永远是自己，对面也是自己先手的，而且随机数据产生在本地，并不互通。
